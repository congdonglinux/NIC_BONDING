Hướng dẫn dùng script 
====================================

- Cài đặt  gói git :
```
  apt-get install git -y || yum install -y git
```

- tải  file về :
```
git clone https://github.com/vdcit/NIC_BONDING
```

- Sử dụng script :
```
cd NIC_BONDING
chmod +X *.sh
```

- Trên ubuntu :
```
  bash Bonding_ubuntu.sh
```
- Trên centos :
```
 bash Bonding_centos.sh
```
